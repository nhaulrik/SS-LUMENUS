## 📋 Idea Evaluation Framework

When considering which ideas to pursue:

### Ask These Questions
1. **User Value**: Does this solve a real user problem?
2. **Effort**: How much work is required?
3. **Impact**: How many users will benefit?
4. **Dependencies**: What needs to be done first?
5. **Maintenance**: How much ongoing work?

### Scoring (1-5 scale)
- **High Value + Low Effort**: Pursue immediately
- **High Value + Medium Effort**: Prioritize for next phase
- **High Value + High Effort**: Plan for future
- **Low Value + Any Effort**: Defer or reject
